# Role Selection

`all_tokenized()` selects all
[`token`](https://textrecipes.tidymodels.org/dev/reference/tokenlist.md)
variables, `all_tokenized_predictors()` selects all predictor
[`token`](https://textrecipes.tidymodels.org/dev/reference/tokenlist.md)
variables.

## Usage

``` r
all_tokenized()

all_tokenized_predictors()
```
